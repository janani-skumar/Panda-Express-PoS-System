# Kitchen Display System

The kitchen display system (KDS) shows real-time orders to kitchen staff, enabling efficient preparation and order fulfillment.

---

## Kitchen View

**Navigate to:** `/employee/kitchen`

The kitchen display shows:

- **Pending Orders** - Orders waiting to be prepared
- **Order Details** - Items in each order with special instructions
- **Order Timing** - Time since order was placed
- **Completion Controls** - Mark items/orders as complete

---

## Order Display

Each order card displays:

| Information | Description |
|-------------|-------------|
| Order Number | Unique identifier for the order |
| Order Time | When the order was placed |
| Items | List of all items to prepare |
| Status | Pending, In Progress, or Complete |

---

## Order Management

Kitchen staff can:

- View all active orders in a carousel format
- See individual items within each order
- Mark items as cooked/complete
- Track order timestamps for preparation timing
- Handle multiple orders simultaneously

> **Note:** The display automatically updates as new orders come in.

---

## Completing Orders

### Step-by-Step Process:

1. **View Order**
   - See all items needed for the order
   - Check for any special instructions

2. **Prepare Items**
   - Cook each item according to standard recipes
   - Maintain food safety standards

3. **Mark Items Complete**
   - Click on individual items when finished
   - Items turn green when marked complete

4. **Complete Order**
   - Once all items are ready, mark entire order complete
   - Order moves to completed queue

---

## Order Priority

Orders are displayed in chronological order (oldest first). Pay attention to:

- **Red indicators** - Order is taking too long
- **Yellow indicators** - Order approaching time limit
- **Green indicators** - Order is on time

---

## Tips for Kitchen Staff

- Start with the oldest orders first
- Communicate with team members about large orders
- Keep work stations clean and organized
- Monitor cooking times closely
