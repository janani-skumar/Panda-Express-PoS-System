# Panda Express PoS System Documentation

![Panda Express Logo](/Panda%20Express/round_logo.png)

Welcome to the Panda Express Point of Sale System documentation. This comprehensive guide covers all aspects of the system including:

- **Customer Ordering** - Self-service kiosk interface with AI-powered chat assistance
- **Cashier Operations** - Employee interface for processing orders
- **Kitchen Display** - Real-time order management for kitchen staff
- **Manager Dashboard** - Administrative tools for inventory, employees, recipes, and reporting

This documentation includes both visual guides and detailed API references.

## System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Stable internet connection
- For employees: Valid login credentials
- For kitchen display: Large screen recommended
