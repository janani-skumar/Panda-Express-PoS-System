# Manager Dashboard

The manager dashboard provides administrative tools for inventory management, employee management, recipe management, and reporting.

---

## Accessing Manager Tools

**Navigate to:** `/employee/manager`

Managers can access the administrative dashboard after logging in with manager credentials.

---

## Dashboard Tabs

The dashboard is organized into four main sections:

| Tab | Description |
|-----|-------------|
| **Inventory** | Stock management and tracking |
| **Employees** | Staff management |
| **Recipes** | Menu item configuration |
| **Reports** | Business analytics and reporting |

---

## Inventory Management

The inventory tab allows managers to:

- **View Stock Levels** - See current quantities of all ingredients
- **Add Inventory** - Record new shipments and supplies
- **Remove Inventory** - Track usage and waste
- **Set Thresholds** - Configure low-stock alerts
- **Generate Restock Reports** - See what needs to be ordered

### Inventory Table Columns:
- Item Name
- Current Quantity
- Unit of Measure
- Minimum Threshold
- Last Updated
- Actions (Edit/Delete)

---

## Employee Management

The employees tab provides:

- **View Staff** - See all employee records
- **Add Employee** - Create new employee accounts
- **Edit Employee** - Update information and credentials
- **Assign Roles** - Set permissions (Cashier, Kitchen, Manager)
- **Deactivate Accounts** - Disable access without deleting records

### Employee Roles:
| Role | Access Level |
|------|--------------|
| **Cashier** | POS terminal, order processing |
| **Kitchen** | Kitchen display, order completion |
| **Manager** | Full dashboard access, reports |

---

## Recipe Management

The recipes tab allows:

- **View Menu Items** - See all available recipes
- **Add Recipe** - Create new menu items
- **Edit Recipe** - Modify ingredients, prices, availability
- **Set Seasonal Items** - Configure limited-time offerings
- **Manage Categories** - Organize items by type

### Recipe Information:
- Name and Description
- Category (Entree, Side, Appetizer, Drink)
- Price
- Ingredients and quantities
- Availability status
- Seasonal dates (if applicable)

---

## Reports

Available reports include:

### Sales Reports
- **X-Report** - Current shift sales summary
- **Z-Report** - End-of-day sales report (closes register)

### Inventory Reports
- **Product Usage** - Ingredient consumption tracking
- **Restock Report** - Items below minimum threshold

### Analytics
- **Sales by Item** - Item popularity analysis
- **Revenue Trends** - Historical sales data

---

## Generating Reports

1. Navigate to the Reports tab
2. Select the report type
3. Choose date range (if applicable)
4. Click "Generate Report"
5. View on screen or export to file

> **Tip:** Run Z-Report only at end of day as it closes out the register.

---

## Best Practices

- Review inventory levels daily
- Update recipes when prices change
- Generate X-Reports during shift changes
- Keep employee information current
- Archive old reports for record keeping
